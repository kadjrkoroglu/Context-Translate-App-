A Flutter translation app powered by Gemini AI that provides **three contextual variations** (Standard, Formal, Casual) for every translation. Features voice input via speech-to-text, auto dark/light theme, and support for multiple languages. Built with clean architecture using Provider for state management and async API integration.

## Translation Modes
The app supports **two translation engines** with seamless switching:
- **Gemini AI Translation**: Advanced contextual translations with three style variations (Standard, Formal, Casual)
- **Offline Google ML Kit**: Fast, offline translation using Google's ML Kit for immediate results without internet connection

Users can easily toggle between these two modes based on their preferences and connectivity needs.

**Tech Stack**: Flutter, Dart, Gemini 1.5 Flash API, Google ML Kit, Speech-to-Text, Provider
